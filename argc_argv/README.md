# holbertonschool-low_level_programming

argc_argv
Description
This directory contains C programming exercises focused on command-line arguments. Learn how to work with argc (argument count) and argv (argument vector) to create programs that accept and process user input from the command line.
Concepts Covered

argc (argument count): The number of command-line arguments passed to a program
argv (argument vector): An array of strings containing the command-line arguments
Parsing and processing command-line parameters
Error handling for missing or invalid arguments
Mathematical operations with command-line input

Exercises
0-whatsmyname.c
Objective: Display the name of the program itself
Print the name of the program as it was invoked from the command line.

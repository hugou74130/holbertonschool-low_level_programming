# holbertonschool-low_level_programming
Description
This directory contains introductory C programming exercises that cover the compilation process, basic output functions, and fundamental C concepts. Start here if you're new to C programming and want to understand how programs are compiled and executed.
Concepts Covered

The C compilation process (preprocessing, compiling, assembling, linking)
Using different compilers and flags
Basic output functions (puts(), printf())
Program structure and syntax
Data types and their sizes
ASCII art and string output

Exercises
0-preprocessor
Objective: Run the C preprocessor
This script runs the preprocessor on a C file. The preprocessor handles #include and #define directives, expanding macros and including header files.

# holbertonschool-low_level_programming
Description
This directory contains C programming exercises focused on creating and using functions combined with nested loop structures. Master function definition, function calls, parameters, return values, and implement complex patterns using nested loops.
Concepts Covered

Function declaration and definition
Function parameters and return values
Nested loop structures
Pattern generation
Loops within functions
Mathematical operations
Character manipulation

Exercises
0-putchar.c
Objective: Print a custom message using putchar
Create a program that prints "_putchar" using only the putchar() function. This teaches the basics of character output.
1-alphabet.c
Objective: Print the alphabet in lowercase
Write a function that prints the alphabet from 'a' to 'z', one character per line.
2-print_alphabet_x10.c
Objective: Print the alphabet 10 times
Create a function that prints the lowercase alphabet 10 times, each on a new line.
3-islower.c
Objective: Check if a character is lowercase
Implement a function that checks if a given character is a lowercase letter. Return 1 if true, 0 if false.

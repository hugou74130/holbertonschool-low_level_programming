This project is part of the **Low-Level Programming** track at **Holberton School**.  
It focuses on understanding **functions**, **nested loops**, and how to organize code for reusability and efficiency.

You will learn how to:
- Define and use functions in C.  
- Work with **nested loops** (loops inside loops).  
- Implement reusable code blocks using headers and function prototypes.  
- Understand how functions communicate using return values.  
- Apply mathematical logic with loops and conditions.

## ‚öôÔ∏è Compilation

All programs are compiled using the following flags:

```bash
gcc -Wall -Werror -Wextra -pedantic *.c -o program_name
```

Example:
```bash
gcc -Wall -Werror -Wextra -pedantic 0-putchar.c _putchar.c -o _putchar
./_putchar
```
